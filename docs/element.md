# 银行卡名词常识

## 卡bin

### 什么是BIN码？

BIN码即银行识别码（Bank Identification Number，简称BIN）通常为银行卡号的前6至8位数字，是国际标准组织分配的用于唯一标识发卡机构的代码。它包含了发卡银行的标识、卡片类型（如借记卡或信用卡）以及所属的支付清算网络（如银联、Visa、Mastercard等）。掌握BIN码信息有助于在支付与结算场景中准确识别卡片属性。

## 还款

信用卡还款是指持卡人在账单到期日前将消费资金返还给银行以维护信用记录的行为，主要方式包括行内还款、移动支付、自助终端及跨行转账等，不同渠道在到账时间和手续费方面存在差异。持卡人需核对账户信息完成资金划转，部分银行支持自动还款设置。逾期可能导致利息、滞纳金及征信受损，恶意透支可被追责。

对于还款，有两个比较重要的时间点：账单日和还款日；而对于还款日没有能力偿还的客户，若不想影响征信，有另外两种方式：分期还款与最低还款。

### 账单日

信用卡消费本质是：先消费，后还款。还款的金额就是上一个账单日到本次账单日间，发生所有有效消费的总额。账单日是固定的，也可以修改。账单日当天的消费为本期消费，但由于信用卡消费需要入账，入账过程需要清算1~2天时间，故账单日当天的金额往往算到下一个账单日。

### 还款日

指的是需要还款的日子，也有“最低还款日”的概念，一般是账单日后18日左右。这段时间就是信用卡还款的“免息期”，产生的账单不需要利息，值得注意的是，信用卡消费入账需要时间，还款入账也需要时间，因此不建议到最后一天还款。

### 最低还款额

最低还款额指的是，信用卡在最后还款日当天，能够在不影响征信的前提下，还款的金额，未还款的金额就会结算利息。

最低还款额**能不碰就不碰**，以其**利息计算复杂**与**利息极高**的特点，成为信用卡“吸血”的最大坑。有人估算过，实用最低还款额后，未还款金额所产生的利息至少在**18%**以上。

例如，以1w还款金额与1k最低还款额为例，利息会计算两个部分：**全额计息（取消免息期）+ 剩余部分** 继续计息，即包括消费日当天开始计算，产生的利息，以及还款日当天开始计算，未还款金额产生的利息，因此利息极高。

\[
\text{总利息} = \underbrace{A \times i \times d1}_{\text{第一部分：取消免息期，全额计息}} + \underbrace{(A - R) \times i \times d2}_{\text{第二部分：未还部分继续计息}}
\]

- **A** = 当期账单全部消费金额
- **R** = 最低还款额（你还掉的钱）
- **d1** = 从**消费日**到**本期还款日**的天数（即原本的免息期长度）
- **d2** = 从**本期还款日**到**下期账单日**的天数
- **i** = 日利率（通常为 **0.05%**）

### 账单分期

账单分期指的是每期账单的还款金额可以进行分期还款，分期还款需要收手续费+利息，利息银行标注的比较明显，但是手续费计算规则复杂，且每期手续费较高，手续费的年利率可能达到13%。

银行信用卡业务营收中，有36.7%来自于账单分期手续费

**注意**：和花呗不同，信用卡提前结清分期会造成违约，不同银行的违约金额计算规则不同

### 容时容差

容时：几天还款宽限期；容差：还款小额差；目前大部分银行容时3天，容差100元。容时容差内，不算逾期。

### 信报合一

信报合一又称信报合并，信报合一就是指你在该银行有多张卡，但在征信报告上只有一个账户。而对于非信报合一的银行，你每申请一张卡，信报上就会多一个账户，这就是很多人发现明明只有几张信用卡，但征信上有几十个账户的原因。

那么**支持**信报合一的银行有：
招商银行、浦发银行、民生银行、华夏银行、平安银行、北京银行、上海银行

那么**不支持**信报合一的银行有：
建设银行、农业银行、中国银行、*工商银行*、交通银行、邮储银行、广发银行、兴业银行、中信银行、光大银行

| 账户<br>分类 | 银行 | 账单规则 | 还款规则 |
| :---: | :---: | :---: | :---: |
| **信报**<br>**合并** | 招商银行、浦发银行、民生银行、华夏银行、平安银行<br>北京银行、上海银行、汇丰银行 | 账单合并 | 还入任意一张卡 |
| **信报不合并** | 建设银行、交通银行、广发银行、中信银行、光大银行、渣打银行、花旗银行 | 账单独立，账单日和还款日统一 | 每张卡单独还款 |
|       ^       | 中国银行、工商银行、农业银行、兴业银行、邮储银行 | 账单独立，账单日和还款日独立 |   ^   |

这里重点标明*工商银行*，一个银行卡可能在征信报告上显示十几个账户（这对征信不会造成影响）
