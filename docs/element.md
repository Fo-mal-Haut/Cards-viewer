# 银行卡基本属性

## 卡bin

### 什么是BIN码？

BIN码即银行识别码（Bank Identification Number，简称BIN）通常为银行卡号的前6至8位数字，是国际标准组织分配的用于唯一标识发卡机构的代码。它包含了发卡银行的标识、卡片类型（如借记卡或信用卡）以及所属的支付清算网络（如银联、Visa、Mastercard等）。掌握BIN码信息有助于在支付与结算场景中准确识别卡片属性。

### 不同卡组织对应的BIN码

|  卡组织  | 对应BIN码 |
| :--------: | :----------: |
|**UnionPay（银联）** |   620000~629999   |
| **Mastercard（万事达）** |  510000~519999   |
| **VISA（维萨）** |   420000~490000   |
|  **JCB**  |     352800~358999     |
|  **American Express（运通）**  |   340000~349999    |
|  **American Express（运通）**  |   370000~379999    |
