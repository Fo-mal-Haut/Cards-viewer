● 完成了。需要在三处地方修改才能实现公式渲染：

  修改内容

  1. build.py — 在 markdown 解析器中添加 \[ ... \] 数学块检测，将公式内容原样输出（不 HTML 转义），包裹在 <div
  class="math"> 中
  2. templates/doc-page.html — 引入 KaTeX（一个轻量的 LaTeX 数学渲染库）的 CSS 和 JS，使用 auto-render
  扩展自动查找并渲染 \[...\] 和 $$...$$ 格式的公式
  3. css/article.css — 为 .math 容器添加居中和横向滚动支持

  原理

  - 之前你的公式行被 escape_html 处理，& 等字符被转义，破坏了 LaTeX 语法
  - 现在公式内容作为纯文本保留在 <div class="math"> 中
  - 页面加载时 KaTeX auto-render 扫描 DOM，找到 \[ ... \] 边界内的 LaTeX 文本，将其渲染为排版后的数学公式
  - 使用 CDN 引入（无额外 npm 依赖），仅对文档页面生效，不增加主页面负载

  重新 python build.py dev 后即可在浏览器中看到渲染效果。公式：

  总利息 = (10,000 × 0.05% × 20) + (10,000 − 1,000) × 0.05% × 11 = 149.5 元

  这一行会以数学排版格式正确显示。
