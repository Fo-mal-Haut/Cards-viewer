# 关于本项目

Cards Viewer 参考了 [GTB](https://github.com/wanglh03) 的开源仓库- [GitHub](https://github.com/wanglh03/cards-viewer)，原仓库是一个用于整理和展示银行卡、信用卡、预付卡和交通卡信息的静态网站。

## 项目说明

本人的理财理念与原作者有部分区别，对于集卡的需求没用那么强烈，银行卡的持有遵循着**少即是多**的理念，因此在原仓库的基础上，去除了“卡片分类”和“卡片排序”等功能。

只保留着部分能够提现本人理财理念的内容，例如：

- 卡片收藏：展示目前持有的银行卡，代表着“现金流”
- 现持信用卡：信用卡是一种消费理念，所以的消费支出都可以走信用卡的渠道
- 文档汇总：一些关于理财、银行的一些关键信息
- 论坛和相关网站：一些可能会用上的实用网站

页面数据和卡面资料由站点维护者持续整理。展示内容仅供信息参考，具体卡片权益、费用及申请条件以发行机构的最新说明为准。
